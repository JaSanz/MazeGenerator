# MazeGenerator
Generador de laberintos en Java

## Idea general del proyecto
El objetivo es crear un generador de laberintos usando diferentes algoritmos y formas.

## Funcionalidades
- Crear laberintos cuadrados de hasta 254 * 254 casillas.
